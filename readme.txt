Gutenberg block editor for be friend with search engines.

== Description ==

Shinobi Blocks is a block library not only useful but also effective.
As you can know, there are many ways to get more trafic to your website. Most known way is "SEO". Of course SEO is definitely the best way, there are other ways to be friendly with search engines. This plugin offers one of these ways that is calles "Structured Data".
For example, this plugin has an "FAQ block". This block looks like making a FAQ, but not only that, it generates data in internally at the same time. Therefore users will be able to see the FAQ in search result page. That means your website is more useful and more unique. Don't you think it's like SEO?

= How to use Shinobi Blocks REST API =

You can use the custom REST API provided by this plugin.

1. https\://your-domain/wp-json/shinobi-blocks/v2/css
1. https\://your-domain/wp-json/shinobi-blocks/v2/structured-data
